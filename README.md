# Portfolio

Personal portfolio site built with Astro and Tailwind CSS.

## Tech Stack

- Astro 6
- Tailwind CSS 4

## Prerequisites

- Node.js 22+
- npm

## Installation

```bash
git clone https://github.com/orfneos/portfolio-astro-app.git
cd portfolio-astro-app
npm install
```

## Usage

```bash
# Development server
npm run dev

# Production build
npm run build
```

Open http://localhost:4321 in your browser.
